# How to install:

1. Download archive
2. Unzip it
3. Replace the data.win file from the game folder with the file from the archive
4. Replace the joker.ogg file from "mus" folder with the file from archive
5. (optional) Replace the file lang_en_ch1.json from the lang folder with the file from the archive

This is all! Now get to Jevil!
